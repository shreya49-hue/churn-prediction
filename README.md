# Customer Churn Prediction

## Project Overview

This project predicts customer churn using Machine Learning techniques. The goal is to identify customers who are likely to discontinue services based on customer demographics, tenure, contract type, and monthly charges.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Label Encoding
4. Feature Scaling
5. Model Building

   * Logistic Regression
   * Random Forest Classifier
6. Model Evaluation

## Key Insights

* Customers with shorter tenure are more likely to churn.
* Month-to-month contract customers have higher churn rates.
* Monthly charges significantly impact customer retention.

## Model Performance

* Logistic Regression Accuracy
* Random Forest Accuracy
